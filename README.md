# Overview
This project provides agent scripts for Unity ML-Agents, along with recorded demonstration (.demo) files, CSV files used to generate and record these demonstrations, YAML configuration files for training, and a Python script for computing average similarity.

# Description
The script includes the agent, the car, and information on recording movement trajectories or calculating the average similarity of movement trajectories.

# Requirements
- Mlagents: ml-agents-release_22
- Python: 3.10.1
